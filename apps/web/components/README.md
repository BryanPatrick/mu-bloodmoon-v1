# Components

Organizacao pensada para qualquer pessoa encontrar rapido onde mexer.

- `branding`: logo e elementos de identidade.
- `content`: blocos reutilizaveis de conteudo, como hero de pagina.
- `forms`: campos, formularios e controles reutilizaveis.
- `layout`: header, footer, shells e estruturas de tela.
- `modals`: modais e dialogs.
- `navigation`: menus, dropdowns e navegacao.
- `tables`: tabelas e listagens tabulares.
- `ui`: pecas pequenas de interface compartilhada.

Regra simples: se um componente pertence a uma tela especifica, primeiro tente deixar perto da tela. Se for reutilizado em mais de uma tela, mova para uma destas pastas.

Padrao visual: seguir `docs/design-system.md`.
