- Sistemas Ruud.

- Arvores de talento modernas com skill cards.

## Sistemas descritos no topico

### Aprendizado e uso

- Para aprender uma skill, o jogador clica com o botao direito no livro/orb/pergaminho.
- Para usar no PC, o jogador seleciona a skill na interface de skills e usa o botao direito do mouse.
- Algumas skills aparecem apenas enquanto a arma ou item correspondente esta equipado.
- Algumas skills dependem de quests ou de personagem evoluido.
- No mobile, a interface usa outro fluxo. Nao usar como referencia de UX para o nosso portal sem necessidade.

### Tipos de origem

- Skill comprada em NPC.
- Skill obtida por drop.
- Skill vinculada a arma/item equipado.
- Skill liberada por quest.
- Skill especial de evento/castle siege.
- Skill de pet.
- Skill por sistemas modernos, como Ruud ou talent tree.
