Ventana de informacion del Pentagram (Agregado apartir de la Ex702)

Tabla de elementos se agrega en la ventana de informacion del pentagram.

- La tabla de elementos se añade en la ventana de informacion del pentagram.
- Muestra los elementos mas fuertes y mas debiles contra el pentagram equipado por el personaje.
- "Non-Elemental" de estado se visualiza cuando no se lleva equipado ningun pentagram.
- Todos los elementos se mostraran como "Maximum contrary element" cuando se esta en "Non-elemental" estado.
- Usted puede comprabar haciendo click en "pentragram information windows".
- Puede comprobar las estadisticas de sus errtels aplicados en el pentragram.
*El ataque y defensa puede ser diferente dependiendo de las estadisticas del personaje.

Eliminación del Errtel
Es Posible que al arrastrar el Errtel a la ventana del Pentagram este se elimine.
Hay una cierta probabilidad de que el Errtel desaparezca.

 

Fortalecimiento del Errtel
Subir el Rank del Errtel es posible cada vez que se suba un Rank con éxito una opción se agregara.
Se puede agregar hasta 3 Rank actualmente (en el futuro serán hasta 5 Ranks).
El "Spirit Powder" es el ítem necesario para subir el Rank y dropea en los BOSS de Elementos de Acheron.
Si el Errtel sube de Rank el Nivel del Errtel bajara a +0
Si la combinación falla los items desaparecerán.

 

Subir el Rank del Errtel
Se puede subir el Rank del Errtel hasta Rank 3
Para subir el Rank del Errtel es necesario ir al NPC "Adniel".

Cuando se sube el Rank del Errtel y agrega una misma opción no es posible tenerlo duplicado.
