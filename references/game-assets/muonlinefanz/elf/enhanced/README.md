# Enhanced item versions

Esta pasta fica reservada para as versoes melhoradas dos equipamentos da Elfa.

Fluxo combinado:

- manter `../original` como referencia bruta;
- nao editar nem remover marcas d'agua das imagens originais;
- criar aqui novas imagens limpas, mais bonitas e mais legiveis;
- preservar silhueta, cor, tier e essencia visual do item original;
- ignorar barras, faixas, brilhos de buff, indicadores de XP e efeitos temporarios de interface presentes em capturas in-game;
- tratar brilho forte de equipamento como estado blessado/upado (+1, +2 etc.), nao como visual base do item;
- em versoes +15, linhas/arcos/chifres brancos sao aura de upgrade maximo, nao parte fisica fixa da armadura;
- preencher `enhancedPath` no `../manifest.json` quando uma versao melhorada for aprovada.

Nota: a barra diagonal vermelha/branca observada em uma referencia enviada e buff de XP do jogo, nao parte do equipamento.
Nota: em versoes neutras, usar brilho controlado; reservar glow intenso para imagens explicitamente blessadas/upadas.
